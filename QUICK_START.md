# 🎯 快速开始指南 - Elon Musk Tweet 预测系统

## 5 分钟快速上手

### 第一步：安装依赖（1分钟）

打开终端，运行：

```bash
pip3 install requests beautifulsoup4 numpy scipy pytz
```

### 第二步：运行程序（10秒）

```bash
cd /Users/tony777/polymarket-predictor
python3 elon_predictor_enhanced.py
```

### 第三步：选择模式（10秒）

```
请输入选项 (1/2/3，默认1): 1
```

按 Enter 键选择交互式菜单。

### 第四步：自动抓取并预测（1分钟）

```
请选择操作 (1-6): 1
```

系统会自动：
1. 访问 XTracker 获取数据
2. 计算多种预测
3. 给出下注建议
4. 保存到历史记录

### 第五步：查看建议（30秒）

你会看到完整的推荐报告：
- ✅ 主推荐区间
- 📊 预测数据
- 🎯 置信度
- 💡 行动建议

- 市场开始后：每 6-12 小时

- 市场后期：每 12 小时

使用"查看历史预测"功能，对比多次预测：
```
选择操作: 3  # 查看历史预测
```

观察趋势：
- ↗️ 上升 → 考虑更高区间
- ↘️ 下降 → 考虑更低区间
- ➡️ 稳定 → 按预测下注

不用每次手动搜索：
```

```

浏览器会自动打开当前市场页面。

### 技巧4：持续监控（高级）

如果你不想手动运行：
```

```

系统会每2分钟自动抓取一次并保存。

## ⚠️ 注意事项

### 1. 自动抓取可能失败

如果 XTracker 网页结构变化，自动抓取可能失败。

**解决方案：** 选择"手动输入预测"，然后：
1. 访问 https://xtracker.polymarket.com
2. 查看当前推文数
3. 手动输入到程序

### 2. 时区问题

系统使用 ET（美国东部时区），确保：
- 开始时间：2026-02-05 12:00 PM ET
- 结束时间：2026-02-07 12:00 PM ET

如果你的系统时区不是 ET，程序会自动转换。

### 3. 区间适配

当前配置的区间：
- `<40`, `40-64`, `65-89`, `90-114`, `115-139`, ...

如果是不同市场，需要修改代码中的 `POLYMARKET_BUCKETS`。

## 🆘 遇到问题？

### 问题1：ImportError

```
ModuleNotFoundError: No module named 'requests'
```

**解决：**
```bash
pip3 install requests beautifulsoup4 numpy scipy pytz
```

### 问题2：自动抓取失败

```
✗ 抓取失败: 无法从页面提取推文计数
```

**解决：** 选择手动输入模式

### 问题3：预测值异常

如果预测值看起来不合理（例如几千条）：
- 检查时间窗口配置是否正确
- 检查已过时间是否正确
- 查看历史趋势对比
